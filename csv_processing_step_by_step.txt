Steps to clean the tariff file.
1. Load the file from any path selected using windows pop-up option
2. If it is .zip file, unzip the file. 
3. Read .csv file
4. Read data from file. Segregate the error and correct data rows.
5. Deduplaicate the correct data.
6. Correct the erred data.
7. Deduplicate the erred data
8. Add two datasets(correct and erred - now corrected) data set.
9. Create a processed files folder (if not present already) inside the same folder containing original file.
10. Save data to file in processed files folder.
